# Global BarterNet — Site Vision & Roadmap

## 1. Vision
Global BarterNet is the world's first AI-Mediated Blockchain Resource Exchange Platform — a post-currency economy where AI agents autonomously match resources, negotiate fair trades, and blockchain ensures immutable trust.

## 2. Tech Stack
- Next.js 14 (App Router)
- React 18 with TypeScript
- TailwindCSS with custom iOS-inspired design system
- Framer Motion + GSAP for animations
- Zustand for state management
- Prisma + PostgreSQL for data
- NextAuth for authentication
- Ethers.js for blockchain integration
- Recharts for data visualization
- Socket.io for real-time updates
